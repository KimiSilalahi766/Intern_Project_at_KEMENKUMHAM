KEMUNGKINAN 1: Submit Form Gagal (Ada Error)
----------------------------------------------
CEK:
1. Apakah ada pesan error merah di halaman form?
2. Apakah SEMUA field sudah diisi? (tidak boleh kosong)
3. Apakah gambar sudah di-upload SEMUA (3 gambar)?
4. Apakah ukuran gambar tidak lebih dari 2MB?

SOLUSI:
- Isi semua field yang kosong
- Upload ulang gambar yang belum
- Kompres gambar jika terlalu besar


KEMUNGKINAN 2: Template Word Belum Benar
------------------------------------------
CEK:
1. Apakah file laporanevaluasi.docx ada di writable/filedox/ ?
2. Apakah template punya placeholder format ${...} ?

SOLUSI:
- Buka template Word
- Tambahkan placeholder dengan format: ${JUDUL}, ${Alamat}, dll
- Lihat file CARA_PAKAI.txt untuk daftar lengkap placeholder


KEMUNGKINAN 3: Error di Server
--------------------------------
CEK LOG:
1. Buka folder: writable/logs/
2. Cari file terbaru: log-YYYY-MM-DD.php
3. Buka dengan text editor
4. Cari kata "ERROR" atau "error"
5. Lihat detail error nya

SOLUSI:
- Perbaiki sesuai error yang muncul
- Atau kirim error message ke developer


KEMUNGKINAN 4: Permission Error (Folder Tidak Bisa Ditulis)
-------------------------------------------------------------
CEK:
- Apakah ada error tentang "permission" atau "tidak bisa menulis"?

SOLUSI (Windows):
1. Klik kanan folder 'writable'
2. Properties ‚Üí Security ‚Üí Edit
3. Pilih user Anda
4. Centang "Full Control"
5. OK ‚Üí Apply

SOLUSI (Linux/Mac):
Jalankan di terminal:
    chmod -R 777 writable/
