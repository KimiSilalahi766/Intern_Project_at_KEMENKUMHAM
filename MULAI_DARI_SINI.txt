LANGKAH 1: INSTALL XAMPP
-------------------------
1. Download XAMPP dari: https://www.apachefriends.org/
2. Install (pilih Apache + PHP saja, MySQL opsional)
3. Install di: C:\xampp

LANGKAH 2: TAMBAH PHP KE PATH
------------------------------
1. Klik kanan "This PC" ‚Üí Properties
2. Advanced system settings ‚Üí Environment Variables
3. Cari "Path" di System variables ‚Üí Edit
4. Klik "New" ‚Üí Ketik: C:\xampp\php
5. OK ‚Üí OK ‚Üí OK

LANGKAH 3: INSTALL COMPOSER
----------------------------
1. Download Composer-Setup.exe dari: https://getcomposer.org/download/
2. Install (auto-detect PHP dari XAMPP)
3. Next ‚Üí Next ‚Üí Install ‚Üí Finish

LANGKAH 4: INSTALL VSCODE
--------------------------
1. Download dari: https://code.visualstudio.com/
2. Install
3. Centang "Add to PATH"
4. Finish

LANGKAH 5: CEK INSTALASI
-------------------------
Buka Command Prompt (Win + R, ketik cmd, Enter), lalu ketik:

    php --version

Harus muncul: PHP 8.2.x (cli)

Ketik:

    composer --version

Harus muncul: Composer version 2.x.x

================================================================================
                        ‚ö†Ô∏è TROUBLESHOOTING CEPAT
================================================================================

Problem: "php: command not found"
-----------------------------------
Solusi:
1. Install XAMPP (lihat langkah instalasi di atas)
2. Tambah PHP ke PATH
3. Restart Command Prompt
4. Coba lagi

Problem: Port 8080 sudah dipakai
---------------------------------
Solusi: Ganti port
    php -S localhost:9000 -t public

Lalu buka browser ke: http://localhost:9000

Problem: File tidak ter-download
---------------------------------
Solusi:
1. Cek apakah SEMUA field sudah diisi
2. Cek apakah 3 gambar sudah di-upload
3. Cek log: writable/logs/log-YYYY-MM-DD.php
4. Baca file: PENTING_FILE_HASIL.txt

Problem: Form tidak muncul
---------------------------
Solusi:
1. Pastikan server jalan (ada tulisan "PHP Development Server started")
2. Pastikan buka URL yang benar: http://localhost:8080
3. Clear cache browser (Ctrl+Shift+Delete)
